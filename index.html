Ozon Seller API

Интеграция Ozon работает через back-to-back прокси Apps Script

Браузер не обращается к Seller API напрямую: каждый запрос отправляется на CORS_PROXY, а Apps Script уже подставляет ключи в серверные заголовки.

Если данные не грузятся, проверьте URL опубликованного Apps Script в константе CORS_PROXY и доступ деплоя «Anyone».
